# Contributing to RIOT-rs

Welcome and thanks for your interest in contributing to RIOT-rs! We appreciate all ways of contributing to the project, code and non-code.


## Core information

- We use the git pull request workflow
    - PR's can be submitted as work-in-progress
    - Please indicate a ready-for-review PR with the button in GitHub
    - We use [Conventional Commits](https://www.conventionalcommits.org/en)
- Please check our [Coding Conventions](https://future-proof-iot.github.io/RIOT-rs/dev/docs/book/coding-conventions.html)
- RIOT-rs is dual [licensed](https://github.com/future-proof-iot/RIOT-rs/blob/main/LICENSE) under the Apache-2.0 and MIT license


## Roadmap

We use a [roadmap](https://github.com/future-proof-iot/RIOT-rs/issues/242) to track issues. If you are looking for an issue to contribute, please check our current list of pull requests there.


## Ask for help

We are happy to hear from you and help you. The best way to reach us is to ask on our [Matrix chat room](https://matrix.to/#/#RIOT-rs:matrix.org).
